## ‚ú® Features

### üñ•Ô∏è **Desktop Environment**
- **Boot Sequence:** Realistic Apple boot animation with sound.
- **Lock Screen:** Password-protected entry with customizable wallpaper.
- **Dynamic Dock:** Fully functional taskbar with bouncing animations.

- **Resume Preview:** Built-in PDF viewer with role-switching (Data Analyst vs. Software Engineer).

- **Smart Agent:** Integrated AI chatbot (powered by Google Gemini) that answers questions about my resume, projects, and experience.
- **Context Aware:** Understands specific queries about "UniGo", "Capzora", and more.

## üöÄ Getting Started

Follow these steps to run the portfolio locally.

### **1. Clone the Repository**
```bash
git clone [https://github.com/SaiyamTuteja/Portfolio-v3.git](https://github.com/SaiyamTuteja/Portfolio-v3.git)
cd Portfolio-v3
````

### **2. Install Dependencies**

```bash
npm install
# or
yarn install
```

### **3. Configure Environment Variables**

Create a `.env` file in the root directory and add your Google Gemini API Key:

```env
VITE_AI_API_KEY=your_google_gemini_api_key_here
```

### **4. Run the Development Server**

```bash
npm run dev
```

Open [http://localhost:5173](https://www.google.com/search?q=http://localhost:5173) in your browser.

## üéÆ How to Use

1.  **Power On:** Click the power button to start the boot sequence.
2.  **Log In:** Click your profile to unlock the desktop.
3.  **Explore:**

      * Drag the **Safari** icon or any file to move it.
      * Right-click on the desktop to change the **Wallpaper** or create a **New File**.
      * Ask the **AI Assistant** about my work experience\!
